Download Cytoscape & MiMI Plugin

Download Cytoscape

You can download Cytoscape from here.

Download and Install MiMIPlugin

There are two ways to install MiMI Plugin,intstall from within Cytoscape or download latest plugin jar file. You can use either of them. If you have old version of MiMI Plugin in your system, you need to do plugin Update or first Clean up the old version of MiMI Plugin and then install the latest version. Otherwise, MiMIplugin duplication or version conflict will mess up plugin functionalities.

  • Install from within Cytoscape
    1. Startup Cytoscape->Plugins->Manage Plugins->Available for Install->Network and Attribute I/O->MiMIPlugin
    2. Install
  • Download plugin jar file
    1. Download plugin jar file from here.
    2. Save the jar file into your Cytoscape/plugings/ directory


Update MiMIPlugin

Please make sure no MiMIplugin.jar in yourcytoscape/plugins/ directory. Otherwise delete it.

  1. Startup Cytoscape->Plugins->Update Plugins->Updatable Plugins
  2. Choose MiMIplugin latest version->Update Selected
  3. Re-startup Cytoscape


Clean up Old MiMI Plugin

In order to avoid MiMI Plugin duplication and version conflict, you need to clean up the old plugin versions in your system before install/update latest plugin.

  1. delete MiMIplugin.jar file from yourcytoscape/plugins/ directory
  2. delete directory ".cytoscape" from C:\Documents and Settings\yourloginname\
